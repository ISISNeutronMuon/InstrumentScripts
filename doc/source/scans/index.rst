Scanning Module
===============

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   tutorial
   instrument

The scanning module is designed to produce simple graphs of a single
value (e.g. neutron counts) as a function of another value (e.g. slit
size).  This simple framework can then be extended to support more
complex operations, such as fitting curves or multi-dimensional scans.
